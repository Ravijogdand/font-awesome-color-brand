Font Awesome Color Brand

A list of brand color in SCSS and CSS for Font Awesome or for what you want!

Download View on Github

Get Started

  1. Import Font Awesome in your code, into the tag <head>:
    <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/font-awesome/4.6.1/css/font-awesome.min.css">
    More information at official website of Font Awesome.
  2. Import Font Awesome Color Brand in SCSS or CSS.

    SCSS

    Import the SCSS file in your project:
    @import 'path/font-awesome-color-brand'
    You can use the class directly in the HTML file or the variable color for your SCSS project.

    CSS

    Import the CSS file in your project with the @import:
    @import url('path/font-awesome-color-brand.css');
    or with the <link> tag:
    <link type="stylesheet" href="path/font-awesome-color-brand.css">
  3. Yeah, let's start to color your icon!
